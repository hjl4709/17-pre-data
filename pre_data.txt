1. cmd
dir : 디렉터리
cd : change directory
.. : 상위폴더
=> cd .. : 상위 폴더로 cd
상대경로 ex) C:\Users\PC>python Desktop/test2.py
절대경로 ex) C:\Users\PC\Desktop\test2.py
code . : 선택한 폴더를 작업폴더로 VisualStudio 열기
* 방향키(위아래) 누르면 이전에 입력했던 코드들 바로 사용 가능

2. Visual Studio
ctrl + shift + p : 검색창 (+theme 테마변경)
ctrl + j : 터미널
실행 ex) python test.py
ipynb : jupytor notebook 확장자

+ Jupytor notebook
code 누르기, 코드 입력
shift + alt + ↓ : 밑으로 복붙

3. 숫자형
int : 정수형
float : 실수형
0o : 8진수 (0의 octa)
0x : 16진수

4. 문자형
str : 문자열
escape code : \ (역슬래시)
대괄호 : []
